iframe Tag

The <iframe> tag in HTML stands for "inline frame." It is used to embed another HTML document within the current HTML document. This can be a webpage, a video, a PDF document, or any other type of content that can be displayed in a browser.

Here is the basic syntax of the <iframe> tag: 

<iframe src="URL_of_content_to_embed" width="width_value" height="height_value" frameborder="0" scrolling="auto"></iframe>
