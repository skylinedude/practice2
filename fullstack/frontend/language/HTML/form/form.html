Form Tag

The <form> tag is an essential part of HTML that is used to create interactive forms on web pages. It is used to collect input from users, such as text inputs, checkboxes, radio buttons, and more.
Here is an example of how the <form> tag is used:

1. The <form> tag is used to create a form.
2. The action attribute specifies the URL to which the form data will be submitted when the user clicks the submit button. In this case, it points to /submit-form.php.
3. The method attribute specifies the HTTP method to be used when submitting the form. Here, it is set to POST.
4. Various form elements such as <input>, <select>, and <textarea> are used to collect different types of input from the user.
5. Input fields have attributes like id, name, and required which help in identifying the input, providing a name for the input field, and specifying whether the input is required.
6. The submit button (<input type="submit">) is used to submit the form data, while the reset button (<input type="reset">) clears all form inputs when clicked.
