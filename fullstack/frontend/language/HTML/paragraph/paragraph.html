Paragraph Tag

In HTML, the <p> tag is used to define a paragraph of text. It is a block-level element that creates a new paragraph, causing the content following it to appear on a new line. The <p> tag is commonly used within the <body> section of an HTML document to structure text content into logical paragraphs.
