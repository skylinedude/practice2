Audio Tag

The <audio> tag in HTML is used to embed audio content on a web page. It allows you to play audio files such as MP3, WAV, or OGG directly in the browser without needing a separate plugin. Here's an example of how to use the <audio> tag: 

      <audio controls>
        <source src="audio-file.mp3" type="audio/mpeg">
        <source src="audio-file.ogg" type="audio/ogg">
        Your browser does not support the audio element.
      </audio>

1. The <audio> tag is used with the controls attribute, which adds playback controls (play, pause, volume control, etc.) to the audio player.
2. Inside the <audio> tag, there is a <source> tag that specifies the audio file (audio-file.mp3) and its MIME type (audio/mpeg for MP3 files).
3. The text "Your browser does not support the audio element." is a fallback message that will be displayed if the browser does not support the <audio> tag or the specified audio format.
