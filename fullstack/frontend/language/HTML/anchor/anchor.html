Anchor Tag

Introduction

An anchor tag, also known as an <a> tag in HTML, is used to create hyperlinks in web pages. It allows you to link to another web page, a specific section within the same page, an email address, a file, or any other URL.

Basic Syntax

Here is the basic syntax of an anchor tag:
<a href="URL">Link Text</a>

Attributes

  1. href: This attribute specifies the URL of the page or resource you want to link to.
  2. Link Text: This is the text that will be displayed as the clickable link on the web page.

For example, if you want to create a link to Google's homepage, the anchor tag would look like this:
<a href="https://www.google.com">Visit Google</a>

Internal Links

You can also use anchor tags to create internal links within the same page or to specific sections of a page using the id attribute.
