Image Tag

In HTML, the <img> tag is used to embed images into a web page. Here's the basic syntax of the <img> tag:
<img src="image_url" alt="alternative_text" width="width_value" height="height_value">

Let's break down each attribute:
1. src: This attribute specifies the URL (web address) of the image. It can be a relative or absolute path.
2. alt: This attribute provides alternative text for the image. It is important for accessibility and SEO purposes. Screen readers use this text to describe the image to visually impaired users, and search engines use it for indexing.
3. width: This attribute sets the width of the image in pixels. It is optional and can be used to control the size of the displayed image.
4. height: This attribute sets the height of the image in pixels. Like the width attribute, it is optional and can be used to adjust the image's displayed size.
