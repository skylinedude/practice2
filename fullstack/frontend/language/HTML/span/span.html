Span Tag

Introduction

In HTML, the <span> tag is used to group inline-elements in a document. It does not add any specific meaning to the content it surrounds but is instead used for styling purposes or to target specific parts of the content with CSS or JavaScript.

Example Usage

Here is an example of how the <span> tag can be used:

<p>This is a <span class="highlight">highlighted</span> text.</p>

Result:
This is a highlighted text.
