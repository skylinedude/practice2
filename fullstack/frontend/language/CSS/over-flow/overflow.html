Overflow in CSS

In web development, an "overflow" in CSS refers to how content that overflows its containing element is handled. When the content inside an element is larger than the dimensions specified for that element, overflow properties come into play to determine how the overflow is displayed.

There are several values for the CSS overflow property:

  1. visible: This is the default value. Content is not clipped and may overflow the content box of the element.
  2. hidden: Content that overflows the content box is clipped and not visible.
  3. scroll: Scrollbars are added to the element so that users can scroll to see the overflowing content.
  4. auto: Similar to scroll, but the browser only adds scrollbars when necessary (when content overflows).

Here's an example to illustrate the use of the overflow property: 

            .para {
                width: 200px; /* Set a fixed width for the container */
                height: 200px; /* Set a fixed height for the container */
                overflow: auto; /* Add scrollbars when content overflows */
            }
