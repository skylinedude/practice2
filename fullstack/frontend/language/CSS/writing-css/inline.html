Inline CSS

Inline CSS refers to styling HTML elements directly within the HTML code using the style attribute.
This method allows you to apply specific styles to individual elements without creating a separate CSS file.
Here's an example of how inline CSS is used:

<p style="font-family: Verdana, Geneva, Tahoma, sans-serif; color: brown;  background-color:aliceblue;">this is inline css</p>
