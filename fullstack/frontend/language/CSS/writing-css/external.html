External CSS

External CSS (Cascading Style Sheets) is a method of styling web pages by linking a separate CSS file to an HTML document.
This approach keeps the style definitions separate from the HTML content, making it easier to manage and maintain the code.
Most of the companies use external CSS for the development of their web project.

Here's how you can use external CSS in your web development projects:
Create a CSS File:
Start by creating a separate CSS file with a .css extension. You can name this file anything you like, but it's common
to use a name that reflects its purpose, such as styles.css.

            
inside styles.css file

            body {
                font-family: Arial, sans-serif;
                background-color: #f4f4f4;
              }
              
              h1 {
                color: #333;
              }
              
              .container {
                width: 80%;
                margin: 0 auto;
              }

Link CSS File to HTML: 

            In your HTML file, use the <link> tag in the <head> section to link the CSS file. 

            <head> 
            <link rel="stylesheet" href="styles.css">
            </head>
