Hover in CSS

In CSS (Cascading Style Sheets), the :hover pseudo-class is used to apply styles to an element when a user hovers their mouse cursor over it. This is commonly used to provide visual feedback to users and enhance the interactive experience of a website or web application.

Here's an example of how you can use the :hover pseudo-class in CSS: 

CSS code
.button {
    background-color: #4CAF50;
    border-radius: 5px;
    box-shadow: #4CAF50;
    color: white;
    padding: 10px 20px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
    transition: background-color 0.3s ease;
    cursor: pointer;
    transition: transform 0.3s ease;
}
Styles applied when hovering over the button 
.button:hover {
  background-color: #45a049; // Dark green
  transform: translateX(20px);
}
