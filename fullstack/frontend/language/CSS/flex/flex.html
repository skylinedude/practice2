Flexbox Definition

CSS Flexbox is a layout model that allows you to design more complex and flexible layouts in CSS. It provides an efficient way to distribute space and align items within a container, whether it's a row or a column.

To use Flexbox, you need to define a flex container and then specify how its children (flex items) should behave. Here are some key concepts and properties of Flexbox:

1. Flex Container: To create a flex container, you use the display: flex; or display: inline-flex; property on an element. This property makes the container a flex container and enables the use of Flexbox properties on its children.
2. Flex Direction: The flex-direction property defines the main axis along which flex items are placed inside the flex container. It can be set to row (default), row-reverse, column, or column-reverse.
3. Justify Content: This property aligns flex items along the main axis of the flex container. Common values include flex-start (default), flex-end, center, space-between, and space-around.
4. Align Items: The align-items property aligns flex items along the cross axis of the flex container. It controls the vertical alignment of items when flex-direction is set to row and the horizontal alignment when flex-direction is set to column. Values can be flex-start, flex-end, center, baseline, or stretch (default).
5. Flex Wrap: By default, flex items will try to fit in a single line. If you want them to wrap onto multiple lines based on available space, you can use the flex-wrap property with values like nowrap (default), wrap, or wrap-reverse.
6. Flex Basis, Grow, and Shrink: These properties control how flex items grow, shrink, and initially size themselves in a flex container. flex-basis sets the initial size of an item, flex-grow determines how much it can grow relative to other items, and flex-shrink determines how much it can shrink.
Here's an example of a simple flex layout: 

<style>
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px; /* Just for demonstration */
  border: 1px solid #ccc; /* Just for demonstration */
}

.flex-item {
  width: 100px;
  height: 100px;
  background-color: #f0f0f0;
  margin: 10px;
}
</style>
