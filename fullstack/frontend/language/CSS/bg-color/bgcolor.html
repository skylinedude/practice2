Background color property

In CSS (Cascading Style Sheets), you can set the background color of an element using the background-color property.
This property allows you to specify a color for the background of an element such as a div, paragraph, header, etc.
The color can be specified using color names, hexadecimal values, RGB values, or HSL values. Here are some examples: 

    
using color names

    div {
        background-color: red;
      }

    
using hexadecimal values

    div {
      background-color: #ff0000; /* Red */
    }

            
using RGB values

            div {
            background-color: rgb(255, 0, 0); /* Red */
            }
