Border-radius in CSS

In CSS, the border-radius property is used to create rounded corners for an element's border. It allows you to control the curvature of each corner individually or all corners together. The border-radius property can take one or two values, which represent the horizontal and vertical radii of the border corners, respectively.

Here's how you can use it: 

      
        .border-rounded {
          border-radius: 50px;
        }

        .border-rounded-separate {
          border-radius: 20px 50px;
        }

        .border-rounded-individual {
          border-radius: 15px 70px 20px 50px;
        }
