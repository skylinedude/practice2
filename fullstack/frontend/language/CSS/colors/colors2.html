Introduction to CSS Colors

CSS (Cascading Style Sheets) supports various color formats that you can use to specify colors for text, backgrounds, borders, and other elements on a webpage.

Keyword Colors

CSS supports a set of predefined color names like red, blue, green, black, white, etc.

First Div Block

RGB Colors

RGB (Red, Green, Blue) colors specify the intensity of each color component using decimal values between 0 and 255.

Second Div Block

RGBA Colors

RGBA colors are similar to RGB colors but include an additional alpha value (transparency).

Third Div Block

HSL and HSLA Colors

HSL (Hue, Saturation, Lightness) and HSLA colors allow you to specify colors based on hue, saturation, and lightness.
